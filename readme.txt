This is a very simple plugin for publishing your wordpress posts to facebook on save/update.

== Description ==

This plugin makes use of the latest facebook api (PHP SDK 4.4.0) to implement functionality to publish your wordpress posts on facebook.

== Installation ==

You need a facebook app with the following permissions, reviews by facebook:
* manage_pages
* publish_actions

Also you have to generate a permanent page access token. Click the following link for a tutorial on how to get one: http://stackoverflow.com/questions/17197970/facebook-permanent-page-access-token

After installing the plugin got to Settings > Simple Facebook Publish and enter your app id, app secret and page access token.
Now on the post edit screen you will see an additional meta box with a checkbox. Check this checkbox before saving/updating a post to publish it on facebook.

**ATTENTION:** Your server needs to run at least php 5.4 or later!
